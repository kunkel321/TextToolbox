# TextToolbox
a small text manipulation utility

![Screenshot of TextToolbox](https://github.com/kunkel321/TextToolbox/blob/main/TTBscreenshot.png)

AHK Forum Thread: https://www.autohotkey.com/boards/viewtopic.php?f=83&t=140654

As indicated in the onboard help, this tool is inspired by Tidbit's https://www.autohotkey.com/boards/viewtopic.php?f=2&t=139350 AHK v1 tool https://www.autohotkey.com/boards/viewtopic.php?f=6&t=14515 of the same name.  This tool looks and functions very much the same, but with a couple of additional features.  

I've been using Tidbit's version for 10 years, since he first shared it.   I don't use it frequently enough to justify it always running in the background, so I've been running it from a hotkey assignment in one of my always-running scripts.  I set up the new TextToolbox (TTB) the same way.  The other ahk file is the recommended code to include in your always-running script.
